# دانلود مستقیم فایل بدون فیلترشکن

> **نیازمندی:** پایتون روی سیستم شما نصب باشد.

این ابزار به شما کمک می‌کند فایل‌هایی که لینک دانلود مستقیم آن‌ها با نت ملی دانلود نمی‌شود (مثلاً نیاز به فیلترشکن دارد) را فقط با استفاده از نت داخلی دانلود کنید.

## نحوه دانلود از یوتیوب (تبدیل به لینک مستقیم)

این ابزار لینک یوتیوب را قبول نمی‌کند. برای دانلود ویدیو از یوتیوب بدون فیلترشکن، یکی از دو روش زیر را انتخاب کنید:

### روش اول: استفاده از سایت (یک مرحله)

لینک ویدیو را در سایتی مثل https://app.ytdown.to/ بگذارید تا لینک دانلود مستقیم بگیرید.

### روش دوم: استفاده از بات تلگرام (سه مرحله)

1. لینک ویدیو را به یک بات دانلودی مثل `@RegaYoutube_Bot` بدهید تا ویدیو را دریافت کنید.
2. ویدیو را به یک بات لینک‌ساز مثل `@PremiumFiletoLink_Bot` یا `@file_link_irbot` بدهید تا لینک مستقیم بسازد.
3. لینک مستقیم را با این ابزار دانلود کنید.

## راه‌اندازی

1. این ریپازیتوری را **fork** کنید و نام آن را **تغییر ندهید** (اگر تغییر دهید، باید در `main.py` هم اسم جدید را وارد کنید).
2. ریپو را روی سیستم خود Clone کنید: `git clone https://github.com/soheilditf5-svg/downloader.git`
3. وارد پوشه پروژه شوید: `cd downloader`
4. روی فایل `req.bat` دوبار کلیک کنید تا پیش‌نیازها نصب شوند.

## تنظیمات

فایل `main.py` را باز کنید و دو مقدار را وارد کنید:

- `REPO_OWNER` : نام کاربری گیت‌هاب خودتان (مثلاً `soheilditf5-svg`)
- `GITHUB_TOKEN` : توکن گیت‌هاب

### ساخت توکن گیت‌هاب

به گیت‌هاب بروید:  
`Settings > Developer settings > Personal access tokens > Tokens (classic) > Generate new token (classic)`  
تیک گزینه `repo` را بزنید (کل دسترسی‌های repo را فعال کنید) و سپس توکن را بسازید و کپی کنید.

## نحوه استفاده

1. روی فایل `run.bat` دوبار کلیک کنید.
2. لینک مستقیم فایل مورد نظر را جایگذاری کنید.
3. اگر می‌خواهید فایل در همان پوشه ابزار ذخیره شود، فقط Enter بزنید.  
   در غیر این صورت، نام پوشه دلخواه را وارد کنید.
